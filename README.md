# eton-nvim

My personal Neovim configuration based on [LazyVim](https://github.com/LazyVim/LazyVim).

## 🚀 Features

- **Language Support**: Optimized for **Java** development (via `jdtls`, `java-debug-adapter`, `java-test`).
- **Terminal Integration**: [toggleterm.nvim](https://github.com/akinsho/toggleterm.nvim) for a seamless terminal experience.
- **Database Management**: [vim-dadbod-ui](https://github.com/kristijanhusak/vim-dadbod-ui) for SQL development.
- **Git Integration**: Built-in git signs and diff views.
- **Appearance**: [Catppuccin](https://github.com/catppuccin/nvim) and [Tokyonight](https://github.com/folke/tokyonight.nvim) color schemes.

## ⌨️ Keymaps

| Key | Description |
| --- | --- |
| `<leader>ff` | Find Files |
| `<leader>fg` | Global Search (Live Grep) |
| `<ctrl-\>` | Toggle Floating Terminal |
| `gd` | Go to Definition |
| `gr` | Find References |
| `K` | Hover Documentation |
| `<leader>ca` | Code Action |

## 🛠 Installation

```bash
git clone git@github.com:etonson/eton-nvim.git ~/.config/nvim
nvim
```
